HtStation BBS - Public Files
=============================

Welcome to the HtStation BBS file download area!

This directory contains files available for download using the YAPP protocol.
Use the [F]ILES command in the BBS main menu to browse and download files.

Available categories:
- software/    - Utilities and applications
- documents/   - Manuals and documentation

To download a file:
1. Type 'F' or 'FILES' in the main menu
2. Select the file you want to download
3. Follow the YAPP transfer prompts

73!
